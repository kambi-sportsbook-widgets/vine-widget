# vine-widget

A widget to display a Vine post via the settings configuration

## Configuration example:

__`client-widgets.js`__

```json

...
{
    "order": 1,
    "widgetId": "Seed widget",
    "args": {
        "vine": {
            "title": "FC Barcelona",
            "href": "https://vine.co/v/iJ0OLlz3OlH"
        }
    }
},
...

```
